# abct

abc-tool in MATLAB and Python

## MATLAB

### Dependencies
- MATLAB ≥R2024a

### Installation

#### Simple
- Just add `./abct-matlab/abct` to your MATLAB path.

#### Package
- Run the following command in MATLAB:
  ```matlab
  mpminstall("./abct-matlab/abct")
  ```

### Usage

```matlab
help functionname
```

## Python

### Dependencies
- Python ≥3.10
- numpy
- scipy
- pydantic

### Installation

```bash
pip install abct
```

### Usage

```ipython
import abct

abct.functionname?
```
