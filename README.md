# abct

Analytical brain connectivity toolbox in MATLAB and Pythong

## MATLAB

### Dependencies
- MATLAB ≥R2020

### Installation

#### Simple
- Just add `./abct-matlab/abct` to your MATLAB path.

#### Package
- Run the following command in MATLAB:
  ```matlab
  mpminstall("./abct-matlab/abct")      % direct installation
  ```

### Usage

```matlab
help functionname
```

## Python

### Dependencies
- Python ≥3.10
- numpy
- scipy
- pydantic

### Installation

```bash
pip install abct
```

### Usage

```ipython
import abct

functionname?
```
